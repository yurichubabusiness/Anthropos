# Anthropos

**Anthropos** is a web application that lets users send messages into the future and preserve their legacy on-chain. Inspired by humanity’s historical attempts to leave a mark—from carved stories in stone to the Voyager Golden Record—this app allows anyone to leave a message that can outlast them digitally.

---

## Features

- **Message Submission**  
  Users can write messages and optionally include their name. Messages without a name are automatically marked as `Anonymous`.

- **Media Upload** *(simulated)*  
  Users can add media files after a simulated payment process.

- **Message Feed**  
  View all messages submitted by users. Includes:
  - Name of the sender
  - Message content
  - Timestamp
  - Simulated transaction ID

- **Search and Filter**  
  Users can search messages by:
  - Name
  - Keyword
  - Date
  - Or all fields

- **Report Feature** *(simulated)*  
  Messages can be reported for moderation.

## Installation

1. **Download the HTML file**  
   Save the file as `anthropos.html`.

2. **Open in Browser**  
   Double-click the file or open it with your preferred web browser.

---

## Usage

1. Enter your name (optional) and your message.  
2. Click **Post** to submit your message.  
3. To add media files, click **Add Media Files** and complete the simulated payment.  
4. View messages in the feed below and use the search/filter options.  
5. Copy links to individual messages using the **Copy Link** button.  

---

## Technologies Used

- HTML5 & CSS3  
- Tailwind CSS  
- JavaScript (vanilla)  
- LocalStorage for message persistence

---

## Notes

- The app currently simulates blockchain transactions and media uploads.  
- Messages are stored locally in the browser’s LocalStorage, so clearing the browser cache will remove all messages.  
- The background features an animated image with a blur overlay for a modern aesthetic.
